Installing World Clock OpenTool

  1. Download the OpenTool's jar
  2. Copy the jar to <jbuilder>/lib/ext (where <jbuilder> is your JBuilder directory)
  3. (Re)start JBuilder
  4. Select World Clock in View|Panes

Configuring World Clock OpenTool

  1. Go to the World Clock preference page (Tools|Preferences|World Clock)
  2. Select or type the path to the cities configuration file (See the description of the file in the worldclock-application project)
  3. Restart JBuilder

Uninstalling World Clock OpenTool

  1. Exit JBuilder
  2. Delete WorldClockViewOT.jar from <jbuilder>/lib/ext
