Installing World Clock Plugin

Installing the plugin directly

  1. Download the pluging zip
  2. Unzip the plugin into the Eclipse directory
  3. Restart Eclipse
  4. Go to Window|Show View|Other
  5. Select World Clock in the dialog

Installing from the Update Site

  1. In Eclipse go to Help|Software Updates|Find and Install...
  2. "Search for new features to install", next
  3. Click on New Remote Site
  4. Name: "World Clock Eclipse Update", URL: "https://worldclock-plugin.dev.java.net/nonav/eclipseupdates/"
  5. Finish the first wizard
  6. Select and install the world clock feature
  7. When done, go to Window|Show View|Other
  8. Select World Clock in the dialog

Configuring World Clock Plugin

  1. Go to the World Clock preference page (Window|Preferences|World Clock)
  2. Select or type the path to the cities configuration file (See the description of the file in the worldclock-application project)
  3. Restart Eclipse

Uninstalling World Clock Plugin

  • Remove all traces of worldclock in the features and plugins subdirectory of the Eclipse directory
