Installing World Clock Module

Installing the NBM directly

  1. Download the module's nbm
  2. In Netbeans go to Tools|Update Center
  3. Select "Install Manually Downloaded Modules (.nbm Files)"
  4. Add the downloaded nbm
  5. Continue the wizard until the module is installed
  6. Select WorldClock in the Window menu

Installing from the Update Centre

  1. In Netbeans go to Tools|Update Center
  2. Select the World Clock Module Update Center
  3. Select the world clock module
  4. Continue the wizard until the module is installed
  5. Select WorldClock in the Window menu

Installing the World Clock Module Update Center

Adding the update centre manually

  1. Go to Tools|Options
  2. Switch to the advanced view
  3. Go to IDE Configuration|System|Autoupdates Types
  4. Right click, New|General Update Center
  5. In the wizard type the name: "World Clock Module Update Center"
  6. In the newly added update center properties, type the server URL: "https://worldclock-plugin.dev.java.net/nonav/nbupdates/updates.xml"
  7. Close and you're done

Adding the update centre though the Update Centre NBM

  1. Download the update centre module's nbm
  2. In Netbeans go to Tools|Update Center
  3. Select "Install Manually Downloaded Modules (.nbm Files)"
  4. Add the downloaded nbm
  5. Continue the wizard until the module is installed

Configuring World Clock Module

  1. Go to the World Clock options page (Tools|Options|Miscellanous|World Clock)
  2. Select or type the path to the cities configuration file (See the description of the file in the worldclock-application project)
  3. Restart Netbeans

Uninstalling World Clock Module

  1. Go to Tools|Module Manager, Toys then select worldclockmodule
  2. Click Uninstall
